# Neubrutalism (Anti-Establishment Design) 🌑🛡️

## Overview
A raw, high-contrast, and functional style that rejects traditional design "polish." It uses bold typography, stark colors, and intentional "ugly" elements.

## Key Characteristics
- **Hard Shadows**: Uses large, offset, non-blurred shadows (e.g., `4px 4px 0px #000`).
- **High Contrast**: Pure black (#000) vs. bright, neon colors (Electric Green, Cyber Yellow).
- **Thick Borders**: Bold, 2px to 4px black borders around every element.
- **Aggressive Typography**: Heavyweight fonts like Inter-Black or IBM Plex Mono.

## Implementation (Tailwind)
```html
<button class="bg-[#A7F3D0] border-2 border-black p-4 font-bold shadow-[4px_4px_0px_0px_rgba(0,0,0,1)] hover:shadow-none transition-all">
  EXECUTE COMMAND
</button>
```

## Best For
- Developer Tools / CLI-style dashboards.
- Cyberpunk-themed products.
- Branding for rebellious or experimental products.

## Pros & Cons
- **Pros**: Unforgettable, high readability, highly functional.
- **Cons**: Can feel jarring or "low-budget" if not balanced correctly.
