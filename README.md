# Design Vault 🎨
A clinical repository of modern UI/UX design patterns, architectural styles, and aesthetic trends for 2025-2026.

## Structure
Each folder represents a specific design movement or pattern, containing:
- `STYLE.md`: Technical breakdown, pros/cons, and implementation tips.
- `assets/`: Reference imagery and mood boards.
- `boilerplates/`: Minimal CSS/Tailwind snippets to kickstart the look.
